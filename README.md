# Задание 1



## Задача 1



### Текст задачи



Дерево содержит целые числа. К каждому числу приписать заданную цифру в

начало. MAP





### Алгоритм решения



Программа начинает свое выполнение с функции main. Запрашивается у пользователя количество элементов для дерева, которое сохраняется в значение count с помощью функции readInt. Функция readInt рекурсивно запрашивает ввод до тех пор, пока не будет введено корректное целое число, выводя сообщение «Ошибка: введите целое число» при неверном формате.

Выполняется неявная проверка через List.init: если count меньше или равно 0, функция List.init сгенерирует пустой список, и дальнейшая работа продолжится с пустым деревом.

При положительном значении count создается последовательность случайных целых чисел в диапазоне от -150 до 160 с помощью List.init и генератора случайных чисел Random().Next. Полученный список сохраняется в значение sourceList и выводится на экран.

Затем список sourceList сворачивается в бинарное дерево поиска с помощью List.fold, применяя функцию insert к каждому элементу, начиная с пустого дерева Nil. Результат сохраняется в значение sourceTree. Функция insert рекурсивно вставляет каждый элемент, соблюдая правило бинарного поиска: меньшие значения уходят в левое поддерево, большие или равные — в правое.

Построенное дерево выводится с помощью функции printTree, которая рекурсивно обходит дерево в симметричном порядке (сначала правое поддерево, затем корень, затем левое), добавляя отступы для визуализации структуры.

Вычисляется сумма элементов исходного дерева с помощью рекурсивной функции sum, которая обходит все узлы и накапливает результат. Сумма выводится на экран.

Запрашивается у пользователя цифра для приписывания с помощью readInt. Введенное значение сохраняется в digit.

Создается новое дерево newTree путем применения функции map к исходному дереву sourceTree. Функция map рекурсивно обходит дерево и применяет к каждому значению функцию prependDigit, которая приписывает заданную цифру в начало числа. Для определения количества разрядов используется рекурсивная функция countDigits, затем вычисляется множитель как 10 в степени количества разрядов, и формируется новое число с учетом знака.

Новое дерево выводится с помощью printTree, после чего вычисляется и выводится его сумма.

Программа завершает работу с кодом возврата 0.

## Задача 1



### Текст задачи



Сколько в дереве, содержащем натуральные числа, элементов заданной

значности (например, трёхзначных)? Значность задается FOLD



### Алгоритм решения



Программа начинает свое выполнение с функции main. Запрашивается у пользователя количество элементов для дерева, которое сохраняется в значение count с помощью функции readInt. Функция readInt рекурсивно запрашивает ввод до тех пор, пока не будет введено корректное целое число, выводя сообщение «Ошибка: введите целое число» при неверном формате. Выполняется неявная проверка через List.init: если count меньше или равно 0, функция List.init сгенерирует пустой список, и дальнейшая работа продолжится с пустым деревом. При положительном значении count создается последовательность случайных целых чисел в диапазоне от -150 до 160 с помощью List.init и генератора случайных чисел Random().Next. Полученный список сохраняется в значение sourceList и выводится на экран. Затем список sourceList сворачивается в бинарное дерево поиска с помощью List.fold, применяя функцию insert к каждому элементу, начиная с пустого дерева Nil. Функция insert рекурсивно вставляет каждый элемент, соблюдая правило бинарного поиска: меньшие значения уходят в левое поддерево, большие или равные — в правое. Вычисляется сумма элементов исходного дерева с помощью рекурсивной функции sum, которая обходит все узлы и накапливает результат. Сумма выводится на экран. Далее программа переходит к выполнению задания по подсчету элементов заданной значности. Запрашивается у пользователя значность числа (количество цифр), которое сохраняется в значение targetLength с помощью функции readInt. Для подсчета количества элементов заданной значности используется функция countByDigitLength, которая принимает целевую значность и дерево. Внутри этой функции определяется локальная функция folder, которая принимает текущий счетчик и значение узла. Для каждого значения вычисляется количество цифр с помощью функции digitCount. Функция digitCount рекурсивно подсчитывает количество разрядов в числе, обрабатывая отдельно случай с нулем. Если количество цифр совпадает с целевой значностью, счетчик увеличивается на 1, иначе остается без изменений. Функция countByDigitLength передает функцию folder и начальное состояние 0 в функцию свертки fold, которая рекурсивно обходит все узлы дерева. Функция fold реализует симметричный обход: сначала обрабатывается левое поддерево с текущим состоянием, затем применяется функция folder к полученному состоянию и значению текущего узла, после чего обрабатывается правое поддерево с новым состоянием. Таким образом, накапливается итоговый счетчик, который возвращается как результат. Полученное количество countByLength выводится на экран с указанием заданной значности. Программа завершает работу с кодом возврата 0.
